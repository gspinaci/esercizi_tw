CSS: Criteri tipografici & Font

Obiettivo: applicare scelte tipografiche richieste.

Esercizio

  1. Apri un nuovo file index.html e incolla il boilerplate qui sotto.
  2. Segui gli ESERCIZI indicati nei blocchi <style> e nelle regole annidate.

Boilerplate da incollare

<!doctype html>
<html lang="it">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Tipografia Web — Playground</title>

  <style>

}

/* ESERCIZIO-1: Scarica e configura un font locale con @font-face
   OBIETTIVO: Scarica un font da 1001fonts.com o Google Fonts.
   Sostituisci "CustomFont" con il nome del tuo font e aggiorna i percorsi.
   Esempio di @font-face corretto:
   
   @font-face {
     font-family: "MioFont";
     src: url("./fonts/MioFont/MioFont-Regular.woff2") format("woff2");
     font-weight: 400;
     font-style: normal;
     font-display: swap;
   }
    Poi modifica il paragrafo corretto. */

/* ESERCIZIO-2: Configura un font da Google Fonts
   OBIETTIVO: Aggiungere un font da Google Fonts (es. Roboto).
   e configura il testo nei paragrafi corretti usando due diversi pesi.
   */

/* ESERCIZIO-3: Modifica colore, tipo e decorazioni del testo
   OBIETTIVO: Crea stili per le classi .rosso, .sottolineato, .maiuscolo:
   
   .rosso { color: #dc2626; }
   .sottolineato { text-decoration: underline; }
   .maiuscolo { text-transform: uppercase; font-weight: 600; }

   Sperimenta con colori diversi, spaziature e trasformazioni del testo */
