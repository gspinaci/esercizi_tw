Esercizio 5: Flex Properties

Obiettivo: Controlla come gli elementi crescono e si restringono con flex-grow, flex-shrink e flex-basis.

Compito: Applica diverse propriet flex agli elementi per controllarne il comportamento.

Applica:

  • flex-grow: 2 all'elemento che dovrebbe crescere di pi
  • flex-basis: 200px all'elemento per impostare larghezza base
  • flex-shrink: 0 all'elemento che non si deve restringere

Osserva come si comportano quando ridimensioni la finestra.

Esercizio 7: Card Layout

Obiettivo: Esempio pratico: layout di card responsive.

Compito: Crea un layout responsive che si adatta a diverse dimensioni dello schermo.

Usa media query e combina flex-direction, justify-content, align-items

  1. Per schermi grandi: imposta flex-direction: row e justify-content: space-between
  2. Per schermi pi piccoli: imposta flex-direction: column e align-items: stretch

Consiglio: use @media (min-width: 768px) per schermi grandi.
